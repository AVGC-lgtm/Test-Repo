# **App Name**: CropSafe Sentinel

## Core Features:

- AI-Powered Photo Analysis: Uses AI to analyze inspection photos, flagging potential counterfeit markers using image recognition. It acts as an automated counterfeit detection tool, assisting field officers.
- Interactive Dashboard: A dashboard view showing key metrics such as inspection counts, seizure rates, and lab sample statuses.
- Streamlined Seizure Logging: Allows field officers to log seizures with details like product info, location, and supporting evidence. Includes timestamping and geolocation.
- Lab Interface Integration: Enables lab coordinators to upload test results and trigger legal alerts for confirmed violations. Uses the TruScan device, and a Gemini Analyzer.
- Inspection Visit Planning: Facilitates the planning of inspection visits by assigning officers, dates, and equipment.

## Style Guidelines:

- Primary color: Forest Green (#3F704D), evoking a sense of nature, safety, and growth, central to agricultural products.
- Background color: Light Beige (#F5F5DC), offering a neutral, clean backdrop that enhances readability and reduces eye strain.
- Accent color: Burnt Orange (#CC6633), drawing inspiration from harvested crops and autumnal fields, it is a natural, rustic color for focus and highlights.
- Clean and readable sans-serif fonts.
- Use of simple and recognizable icons to represent different modules and actions.
- Clear, grid-based layout to ensure information is easily accessible.
- Subtle animations on data updates or new notifications.
